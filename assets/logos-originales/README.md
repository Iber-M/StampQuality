# Logos Originales de Aliados

Coloca aquí los logos reales de las empresas aliadas con los siguientes nombres:

## Nombres de archivos requeridos:

1. `nal.png` - North American Lighting
2. `nidec.png` - Nidec Corporation
3. `taxan.png` - TAXAN México
4. `daikin.png` - Daikin
5. `continental.png` - Continental
6. `plastics_technology.png` - Plastics Technology
7. `plastic_omnium.png` - Plastic Omnium
8. `integrity_tool.png` - Integrity Tool & Mold
9. `asiaway.png` - Asiaway Automotive
10. `arcosa.png` - Arcosa Industries

## Formato recomendado:
- **PNG** con fondo transparente
- **Resolución:** Mínimo 300x120px, máximo 600x240px
- **Calidad:** Alta resolución para web

Una vez que coloques los logos aquí, avísame para copiarlos a la carpeta `assets/logos/` y actualizar la página web.
